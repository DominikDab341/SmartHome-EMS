# ZZPO Project

## 🚀 Pakiety i Środowisko
1. **Frontend:** React + TypeScript
2. **Backend:** FastAPI
3. **Baza danych:** PostgreSQL
4. **ORM i Migracje:** SQLAlchemy 2.0 (asyncpg), Alembic

## 🌐 Linki do narzędzi deweloperskich

- 💻 **Frontend:** [http://localhost:5173](http://localhost:5173)
- 🔌 **Backend:** [http://localhost:8000](http://localhost:8000)
- 📖 **Backend (Dokumentacja Swagger/OpenAPI):** [http://localhost:8000/docs](http://localhost:8000/docs)
- 📡 **Baza Danych (PostgreSQL):** `localhost:5432`
- ⚕️ **Status Bazy / Health Check:** [http://localhost:8000/health](http://localhost:8000/health)

## 🛠️ Podstawowe Komendy

Uruchamiania całego środowiska developerskiego:
```bash
docker-compose up -d --build
```

Zatrzymywanie środowiska:
```bash
docker-compose down
```
